# 📓 Notebooks

**This is not an official Google product.**

A collection of Jupyter notebooks exploring Google Cloud AI, ML, and developer tools.

| Notebook | Description |
|:---------|:------------|
| [🕵️ Gemini Deep Research Agent](deep_research.ipynb) | Autonomously plan, execute, and synthesize multi-step research tasks using the Gemini Deep Research agent and the Interactions API. |
| [🚀 Cloud Run LLM Serving with Cloud Storage FUSE](cloud_run_llm_serving_with_cloud_storage_fuse.ipynb) | Deploy large language models on Cloud Run, using Cloud Storage FUSE for efficient model management. |
| [📦 Google Drive ZIP to GitHub Exporter](zip-to-repo.ipynb) | Push the contents of a ZIP file from Google Drive into a new or existing GitHub repository. |
| [🔍 Identifying LLM "Tells"](detecting_ai_text_signatures.ipynb) | N-gram analysis to identify words and phrases statistically more likely in AI-generated text vs. human writing. |
| [💻 Querying a Codebase with Vertex AI RAG](rag_codebase.ipynb) | Index code files from a public GitHub repo and ask questions about the codebase using Vertex AI RAG Engine. |
| [🛍️ Product Data Enrichment with Vertex AI](Product_Data_Enrichment_with_Vertex_AI.ipynb) | Enrich product description metadata using Generative AI to improve engagement and conversion rates. |
| [📈 Causal Inference with Vertex AI Forecasting](causal_inference_with_vertex_ai_automl_forecasting.ipynb) | Estimate the effect of an intervention using tfcausalimpact and Vertex AI AutoML Forecasting. |
| [🏥 Medical Imaging Pipeline](medical_imaging_pipeline.ipynb) | Pre-process DICOM medical images, create an AutoML model, and deploy it to an endpoint using a Vertex AI pipeline. |
| [🏥 Medical Imaging Custom Training](medical_imaging_custom_training.ipynb) | Train a TensorFlow model on medical images using a Vertex AI managed dataset. |
| [🧠 TensorFlow SHAP Explainability](tensorflow-shap-college-debt.ipynb) | Build a tf.keras model and analyze feature importances using the SHAP library. |
| [📰 20 Newsgroups AutoML Import](20_newsgroups_automl.ipynb) | Download and transform the 20 newsgroups dataset into a CSV for Google Cloud AutoML Natural Language. |
| [💬 Google Cloud Natural Language API](google_cloud_natural_language_api.ipynb) | Entity extraction, text classification, sentiment analysis, and syntax analysis with the Natural Language API. |
